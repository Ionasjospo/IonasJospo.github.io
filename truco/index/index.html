Aprende a jugar al truco

Les doy la bienvenida a la mejor página para aprender a jugar al truco. Aquí podras aprender como jugar usando las diferentes secciones. Podras simular una ronda para ver tus cartas, conocer como se forma las flores y los envidos. Tendras una calculadora para verificar los puntos de estos. Pulsa dentro de las tarjetas para acceder a las secciones!

Cartas

En esta sección, conoceras las diferentes cartas del juego y sus jerarquia. Tambien puedes simular tu ronda para ver todas las cartas de tu partida!.

Flor

En esta sección, conoceras los diferentes tipos de Flor del juego. Tambien tendras una calculadora para conocer cuanto vale tu flor.

Envido

En esta sección, conoceras los diferentes tipos de envido que hay en el juego. Tambien tendras una calculadora para conocer cuanto vale tu flor.

Señas

En esta sección tendras los diferentes tipos de señas del juego para poder jugar en equipo..
